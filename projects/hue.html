Control My Lights!


Above is a livestream of my room and a remote that you can click to manipulate my light bulbs. Depending on your internet speed, the livestream may take some time to update. There is a spam filter if you make too many change requests.

This project was built using the Python library phue and a Raspberry Pi Zero as a self-contained wireless camera. Due to proxy and mixed-content reasons, this page may throw a security flag in your browser.
